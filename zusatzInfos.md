Dev-PC Hardware: RTX 5090 32GB, 64GB RAM, 2TB NVMe SSD, ArchLinux

Prod-Server Hardware: Proxmox, Intel Arc A310 4GB, 16GB RAM


Für schnelle Tests und die Entwicklung von Klara wird ein leistungsstarker Dev-PC mit einer RTX 5090 verwendet. Für den produktiven Einsatz und die Bereitstellung muss Klara so optimiert werden, dass sie auf einem Server mit einem Proxmox-Server und einer Intel Arc A310 Grafikkarte effizient läuft.
